        Описание

        Блок slider («слайдер») — управляющий элемент интерфейса для удобного и наглядного выбора диапазона значений. Позволяет создавать слайдер с одним или двумя ползунками, равномерной или неравномерной шкалой.

        JS-параметры

        • paramsObject

          JS-параметры блока.

        • params.minNumber

          Минимальное значение слайдера. Если не задано, будет использовано крайнее правое значение шкалы.

        • params.maxNumber

          Максимальное значение слайдера. Если не задано, будет использовано крайнее левое значение шкалы.

        • params.scaleObject[]

          Массив объектов, описывающих деления шкалы. Каждый элемент массива - объект со следуюшими ключами:

          • value - Значение элемента шкалы.
          • step - Шаг изменения шкалы на заданном участке.
          • percent - Процент от общей ширины шкалы. Обязателен для всех значений, кроме первого и последнего.
          • label - Метка деления. Будет отображена в нижней части слайдера, в позиции, заданной параметром percent.

        Методы

        toAllowedRange

        PROTECTED

        Приводит значения выставляемое значение к допустимому диапазону.

        Переопределите этот метод для задания отличного от [min; max] диапазона.

        Параметры
        • idxNumber

          Индекс ползунка, для которого выставляется значение.

        • valNumber

          Текущее значение ползунка.

        Возвращаемое значение
        Number

        Значение, приведенное к допустимому диапазону (по умолчанию [min; max]).

        val

        Устанавливает значение первого ползунка, если передан параметр val, или возвращает текущее значение, если параметр не передан.

        Если блок выключен, новое значение не будет уcтановлено.

        Параметры
        • valNumber

          Выставляемое значение.

        • dataObject

          Дополнительные данные, которые будут переданы в качестве параметра сгенерированному событию change.

        Возвращаемое значение
        BEM.DOM, Number

        Текущее значение или сам блок, если значение устанавливается.

        vals

        Выставляет или получает текущее значение для одного или нескольких ползунков.

        Если блок выключен, новые значения не будут установлены.

        Выставить первый ползунок в значение 123:

        slider.vals(0, 123);

        Выставить значения двух ползунков в 123 и 456:

        slider.vals([123, 456])

        Получить текушие значения ползунков:

        slider.vals() //вернет массив [123, 456]

        Получить текущее значение второго ползунка:

        slider.vals(1) //вернет значение 456

        При установке значений, генерирует событие change.

        Параметры
        • idxNumber

          Индекс ползунка для установки или получения значения. Если не передан, то val должен быть массивом, содержащим значения всех ползунков. Индексация начинается с 0.

        • valNumber | Array.

          Значение ползунка для установки. Если передан массив, будут установлены значения для всех ползунков.

        • dataObject

          дополнительные данные, которые будут переданны в обработчик события change.

        Возвращаемое значение
        BEM.DOM, Number, Array.

        Текущее значение одного ползунка, все значения или сам блок, если значение устанавливается или блок заблокирован.

        min

        Устанавливает минимально допустимое значение, если передан параметр min, или возвращает текущее значение, если параметр не передан.

        Параметры
        • minNumber

          Устанавливаемое минимальное значение.

        Возвращаемое значение
        BEM.DOM, Number

        Текущее значение или сам блок, если значение устанавливается.

        max

        Устанавливает максимально допустимое значение, если передан параметр max, или возвращает текущее значение, если параметр не передан.

        Параметры
        • maxNumber

          Устанавливаемое максимальное значение.

        Возвращаемое значение
        BEM.DOM, Number

        Текущее значение или сам блок, если значение устанавливается.

        range

        Устанавливает или возвращает диапазон допустимых значений.

        Параметры
        • minNumber

          Минимально допустимое значение.

        • maxNumber

          Максимально допустимое значение. Если значение не передано или оно меньше min, будет установлено максимально допусимое значение текущей шкалы.

        Возвращаемое значение
        BEM.DOM, Array.

        текущие значения или сам блок, если значение устанавливается.

        isRunnerDisabled

        Возвращает true, если ползунок выключен.

        Параметры
        • runnerNumber | jQuery

          Индекс ползунка или его DOM-элемент.

        Возвращаемое значение
        Boolean

        isDisabled

        Возвращает true, если слайдер выключен.

        Возвращаемое значение
        Boolean

        getInterval

        Возвращает текущий интервал по шкале для активного ползунка.

        Возвращаемое значение
        Number

        getDefaultParams

        Возвращает параметры по умолчанию.

        Возвращаемое значение
        Object

        getOrientation

        Возвращает ориентацию слайдера.

        Возможные значения:

        • vert - вертикальный слайдер
        • hor - горизонтальный слайдер
        Возвращаемое значение
        String

        Ориентация слайдера.

        События

        change

        Генерируется при изменении значения любого ползунка.

        Параметры
        • dataObject

          Дополнительные данные о событии.

        • data.sourceString

          Источник изменения. Принимает значение drag, если значение изменено перетаскиванием ползунка.

        dragend

        Генерируется при завершении перетаскивания ползунка (когда пользователь отпускает кнопку мыши).

        Модификатор _interactive_no

        Слайдер, не выводящий значения на ползунках.

        Модификатор _orientation_horiz

        Горизонтальный слайдер.

        Модификатор _type_range

        Слайдер с двумя ползунками, предназначенный для выбора диапазона значений.
