Тумблер

Описание

Блок tumbler («тумблер») служит для выбора одного из двух предложенных вариантов. Визуально представлен переключателем с двумя положениями «Вкл/Выкл». Текст, расположенный на тумблере «Вкл/Выкл» – неизменяемый и локализованный (значение устанавливается из Танкера). Функциональность тумблера основана на двух нативных радиокнопках <input type="radio" атрибуты>, которые скрыты.

В MSIE 7-8 блок отображается простыми прямоугольниками со сплошной заливкой. При выключенном JS – деградирует в две нативные радиокнопки.

Объявление блока на странице

BEMJSON для использования тумблера:

{
    block: 'tumbler',  // имя блока
    name: 'color',     // имя для идентификации блока. Не обязательное
    mods: {size: 's'}, // модификаторы блока. Модификатор размера – обязательный
    content: [ // левая и правая метки тумблера. Не обязательное поле
        { //создание метки для левой радио-кнопки
            elem: 'option',
            side: 'left',      // левая метка
            content: 'серый',  // текст метки слева
            value: 'gray'      // значение для левой радиокнопки. Не обязательное
        }
        { //создание метки для правой радио-кнопки
            elem: 'option',
            side: 'right',      // правая метка
            content: 'желтый',  // текст метки справа
            value: 'yellow'     // значение для правой радиокнопки. Не обязательное
        }
    ]
}

В простейшем случае (тумблер без меток со значениями по умолчанию):

{
    block: 'tumbler',   // имя блока
    mods: {size: 's'}, // модификаторы блока. Модификатор размера – обязательный
}

Атрибуты блока

Атрибуты тумблера аналогичны атрибутам нативной радиокнопки:

  • {String} name — имя для идентификации. Задается в одноименном поле блока. В HTML для каждой радиокнопки сформируется одинаковый атрибут name с заданным значением. По умолчанию значение name генерируется случайным образом.

  • {String} value – значение, отправляемое на сервер, уникальное для каждой радиокнопки (элемент блока option). Задается в поле элемента option. Если для option явно не задано value, то для левой радиокнопки одноимённый атрибут примет значение false, а для правой — true.

name, value – преобразуются в соответствующие атрибуты нативных радиокнопок.

Содержимое блока

  • {Array} content – используется для создания текстовых меток тумблера, расположенных слева и/или справа от контрола. Тумблер можно переключить (включить/выключить) нажатием левой кнопки мыши на метку. Метки создаются в поле content соответствующего элемента блока option (левой или правой радиокнопки) следующим образом:
{
    block: 'tumbler',
    name: 'color',  // имя для идентификации блока. Не обязательное
    mods: {size: 's'},
    content: [
        { //создание метки для левой радио-кнопки
            elem: 'option',
            side: 'left',      // левая радиокнопка (side:'right' - для правой)
            content: 'серый',  // текст метки слева
            value: 'gray'      // значение для левой радиокнопки. Не обязательное
        }
    ]
}

В результате будет создана метка слева от тумблера.

По умолчанию метки отсутствуют. В HTML преобразуются в тэг <span> (в отличии от радиокнопок – <label>).

Модификаторы

Темы оформления

Тумблер имеет единственную, стандартную тему оформления – обычную (модификатор theme со значением normal). Используется по умолчанию.

Размеры

Для тумблера реализованы два размера - s и m, которые определяют размеры блока и шрифта. Модификатор размера size – обязательный при объявлении блока, по умолчанию не определен. Если во входных данных не установлен – контрол на веб-странице не отобразится.

Если необходим другой размер, его можно доопределить на проекте. Кастомный размер.

Состояния тумблера

  • Активен, модификатор блока checked со значением yes. Переключатель с правой стороны (Вкл). Тумблер может быть включен/выключен — переключатель находится справа/слева, т.е. выбирается правая или левая радиокнопка (в HTML будет добавлен атрибут checked для соответствующей радиокнопки).

  • Не активен, модификатор блока disabled со значением yes. Тумблер не доступен для действий пользователя (в нативных радиокнопках будет выставлен атрибут disabled). Метки тумблера, при наличии, тоже будут не активны.

По умолчанию тумблер создается в состоянии активен и выключен — переключатель с левой стороны (атрибут checked установлен для левой радиокнопки).

Переключить тумблер можно с помощью клавиш Space и [ ←, ↑, →, ↓ ].

Значения по умолчанию

  • Тема оформления: обычная, модификатор theme со значением normal.
  • Состояние тумблера: активен и выключен (переключатель с левой стороны).
  • Текст на тумблере «Вкл/Выкл» – неизменяемый и устанавливается из Танкера.
  • Если не были явно заданы:
    • name — значение сгенерируется случайным образом;
    • value – для левой радиокнопки примет значение false, а для правой — true.
