Описание

Контроллер для связи блоков, от которых нужно показать сообщение, с блоком alert. Связь выполняется с помощью блока alert-message, который подмешивается к блоку-источнику сообщения.

Сообщения могут быть трех типов:

  1. info - информационное сообщение;
  2. error - сообщение об ошибке;
  3. block - блокирующее сообщение.

Методы

show

Генерирует и показывает попап (блок popup) с сообщением в блоке alert. Перед показом скрывается саджест, положение попапа и время его показа зависит от типа сообщения.

Параметры
  • typeString

    Тип сообщения, возможные значения info|error|block.

  • paramsString

    Передаваемые данные.

  • params.messageString

    Сообщение для отображения.
