Описание

Управляемое выпадающее окно.

Является составным блоком, объединяющим выпадающее окно (popup) и вызывающий его элемент, относительно которого оно открывается.

Служит для создания элементов веб-интерфейса, используемых для скрытия дополнительной информации, ненужной в данный момент. Например, разного рода выпадающие меню, подсказки и т.п.

Элементом управления является элемент switcher.

switcherBlock – имя блока, который нужно найти на элементе switcher (на одном узле может быть несколько блоков). Если switcherBlock не задан, то popup показывается/скрывается при клике левой кнопкой мыши в любом месте элемента switcher.

Если же он задан, то открываться/закрываться popup будет при клике на блок switcherBlock внутри элемента switcher.

Методы

getPopup

Возвращает инстанс popup-а.

Возвращаемое значение
BEM.DOM

getDefaultParams

PROTECTED

Декларация методов блока popup, доступных для выполнения в блоке dropdown.

Возвращаемое значение
Object

Возвращает объект, содержащий имя элемента и методы popup.

destruct

PROTECTED

Удаляет блок, предварительно удалив слушатели событий блока popup.

События

show

Генерируется, когда блок popup отображается.

hide

Генерируется, когда блок popup скрывается.
