Абстрактная иконка

Описание

Блок icon – абстрактная иконка. Используется внутри других блоков в качестве контейнера для декоративных изображений. Представляет собой HTML-тэг <i> с фоновой картикой, заданной через background-image.

Объявление блока на странице

BEMJSON для использования блока:

{
    block: 'icon'
}

Варианты использования блока

Путь к изображению, ширина и высота иконки задаются с помощью css-свойств background-image, width и height соответственно.

Свойства иконки можно задать в модификаторе блока icon:

.icon_type_active
{
    width: 16px;
    height: 16px;

    background-position: -16px 0;

    backgroun-image: url(image-sprite.png);
}

Также можно задать свойства иконки каскадом в родительском блоке или его элементе:

.button_icon_comment .icon
{
    width: 16px;

    background-image: url(icon-comment.png);
}
