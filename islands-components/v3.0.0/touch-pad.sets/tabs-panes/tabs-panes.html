Панельки для табиков

Внимание! Блок имеет нестабильный API

Панельки для табиков, реализуют переключение контента. Может работать с любым блоком, который геренинует событие change и передает в качестве параметров {Number} сurrentIndex и {Number} prevIndex. Пример такого блока - блок tabs.

Чтобы связать панельки с другим блоком, к последнему нужно примешать [{block: 'tabs-menu', js: {id: 'your-binding-id'}}] и использовать этот же id в объявлении tabs-panes; также нужно доопределить статический метод getSupportedControls, чтобы он возвращал названия нужных блоков. Поумолчанию, возвращает tabs.

Пример подключения:

{
    block: 'my-custom-block',
    mix: [{block: 'tabs-menu', js: {id: 'your-binding-id'}}]
    // ...
},
{
    block: 'tabs-panes',
    id: 'your-binding-id'
    // ...
},

Элементы

pane

Панелька, скрыта в обычном состоянии, показана с модификатором `tabs-panes__pane_active_yes`.
