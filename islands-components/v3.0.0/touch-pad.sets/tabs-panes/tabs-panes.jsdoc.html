Методы

findPane

Возвращает jQuery-коллекцию элементов панелек.

Рекомендуется исаользовать именно этот метод для получения панелек, т.к. он возвращает только панельки, относящиеся к текущему блоку. Если вложить tabs-panes в другой tabs-panes, то при прямом вызове tabsPanes.elem('pane') будут возвращены панельки внешнего и внутреннего блоков.

Параметры
  • modNameString
  • modValString
Возвращаемое значение
jQuery

STATIC

Возвращает список доступных блоков, способных работать в качестве контрола-переключателя для панели. Доопределите этот метод для добавления своих блоков.
