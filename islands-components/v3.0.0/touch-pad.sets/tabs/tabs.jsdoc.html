      Описание

      Свойства

      getSupportedControls

      Возвращает индекс активного табика или -1, если нет активного табика. Индексация начинается от 0.

      События

      change

      Генерируется при смене активного таба.

      Параметры
      • dataObject

        Данные события.

      • data.currentIndexNumber

        Индекс активного таба.

      • data.prevIndexNumber

        Индекс предыдущего активного таба.
