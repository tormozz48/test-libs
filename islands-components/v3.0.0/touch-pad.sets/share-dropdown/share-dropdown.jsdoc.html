Описание

Блок «Поделиться».

Блок share-dropdown, cлужит оберткой для блока «Поделиться», основан на блоке dropdown-menu. Визуально представлен кнопкой «Поделиться» и выпадающим окном со списком популярных социальных сетей. Позволяет посетителям размещать ссылку на страницу сайта в коммуникационных сервисах. Ссылки открываются в новом окне, строятся в зависимости от сервиса и различных параметров, могут изменяться динамически.

Методы

updateLink

Обновление данных для указанного сервиса.

Параметры
  • serviceString

    Имя сервиса, соответствующее выставленному модификатору (facebook, moimir, etc.).

  • paramsObject

    Параметры, по которым строится ссылка для сервиса.

  • params.serviceString

    Сервис. Если данный параметр не указан, то берется значение data-атрибута data-service из DOM-элемента блока link. Иначе, переданное значение параметра должно соответствовать первому параметру (service).

  • params.urlString

    Адрес страницы, которой необходимо поделиться. Значение по умолчанию берется из window.location.href.

  • params.titleString

    Заголовок страницы. Значение по умолчанию - document.title.

  • params.imageString

    Дополнительная картинка, передаваемая в сервис. Для Pinterest этот параметр является обязательным.

  • params.descriptionString

    Описание ссылки. Если параметр не указан, описания берется из мета-тега description.

Возвращаемое значение
BEM.DOM

Экземпляр блока.
