Описание

Блок checkbox («чекбокс») – независимый переключатель. Позволяет управлять параметром с двумя состояниями – включено и выключено.

Методы

isDisabled

Шорткат для проверки модификатора _disabled_yes.

Возвращаемое значение
Boolean

isChecked

Шоткат для проверки модификатора _checked_yes.

Возвращаемое значение
Boolean

toggle

Хелпер для переключения модификатора _checked_yes.

val

Получить/установить значение контрола.

Параметры
  • valString

    значение которое нужно установить

Возвращаемое значение
String, BEM.DOM

События

change

Происходит после изменения модификатора checked.

Параметры
  • checkedBoolean

    Выставлен ли модификатор checked в значение 'yes'.
