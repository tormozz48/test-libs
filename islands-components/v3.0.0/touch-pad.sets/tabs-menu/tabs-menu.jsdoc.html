Описание

Методы

getActive

Возвращает элемент активного табика.

Возвращаемое значение
jQuery

getTab

Возвращает DOM-элемент табика по его индексу.

Параметры
  • indexNumber

    Индекс табика. Индексация начинается от 0.

Возвращаемое значение
jQuery

indexOfTab

Возвращает индекс табика или -1, если переданный элемент не найден среди табиков.

Параметры
  • elemjQuery

    DOM-элемент табика.

Возвращаемое значение
Number

События

change

Генерируется при смене активного табика.

Параметры
  • dataObject

    Данные события.

  • data.currentjQuery

    Активированный табик.

  • data.prevIndexjQuery

    Деактивированный табик.
