Меню из табиков

Внимание! Блок имеет нестабильное API

Блок, реализующий меню в виде табиков.

Может выполнять роль простого меню, без js-поведения. Для этого просто не нужно устанавливать BEMJSON-параметр js.

Имеет размеры s и m.

Дружит с блоком island.

Если добавить BEMJSON-параметр js: true, то блок приобретает js-функциональность. Подробнее в API.

Модификаторы _size, _theme, layout по умолчанию не выставляются, нужно самостоятельно позаботьтся о них и их зависимостях.
