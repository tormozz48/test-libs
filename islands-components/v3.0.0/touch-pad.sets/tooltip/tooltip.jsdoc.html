    Описание

    Всплывающая подсказка-указатель.

    Методы

    getOwner

    Возвращает объект на который указывает подсказка.

    Возвращаемое значение
    BEM.DOM, jQuery

    setOwner

    Определяет объект на который будет указывать подсказка.

    tooltip
 .setOwner(ownerBlock)
 .setMod('shown', 'yes');
    Параметры
    • ownerBEM.DOM | jQuery
    Возвращаемое значение
    BEM.DOM

    this

    redraw

    Перерисовывает подсказку относительно заданного owner, в случае если она показана.

    Возвращаемое значение
    BEM.DOM

    this

    destruct

    PROTECTED

    События

    click

    Генерируется при клике по подсказке.

    show

    Герерируется при показе подсказки.

    hide

    Генерерируется при скрытии подсказки.

    Модификатор _autoclosable_yes

    Подсказка, автоматически закрывающаяся при клике за ее пределами, а также по нажатию на клавишу ESC.
