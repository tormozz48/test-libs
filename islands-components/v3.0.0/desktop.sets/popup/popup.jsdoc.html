      Описание

      Попап, автоматически закрывающийся при нажатии за его пределами.

      Методы

      _pushTailPos

      Смещает позицию хвостика на указанные во втором аргументе координаты с учётом текущей позиции.

      Параметры
      • keyString

        Уникальный ключ направления

      • offsetObject

        Объект, описывающий позицию для смещения

      Возвращаемое значение
      block

      Свойства

      __base

      Метод, позволяющий выбрать между div и iframe по кастомным факторам.

      События

      show

      Генерируется при показе блока.

      Если включена анимация, то генерируется сразу в момент ее начала.

      hide

      Генерируется при скрытии блока.

      Если включена анимация, то генерируется по ее окончании.

      Модификатор _adaptive_yes

      Попап, автоматически меняющий направление раскрытия при изменении размеров окна браузера или прокрутке.

      Направление выбирается из js-параметра directions таким образом, чтобы на экране отображалась максимум содержимого попапа.

      Методы

      getScrollEvents

      PROTECTED

      Возвращает массив с именами событий скрола. Сделал так для удобного доопределения с уровня тачей.

      Возвращаемое значение
      Array

      onScroll

      Обработчик события scroll. Подчищает кэши, инициирует перерисовку блока.

      Параметры
      • eEvent

      destruct

      Уничтожает блок, выключает адаптивность (отписывается от события scroll на DOM-элементе блока).

      Модификатор _animate_yes

      Анимированный попап.

      При показе и скрытии проигрывает анимацию.

      Методы

      afterShow

      PROTECTED

      Метод предназначен для проигрывания анимации показа блока.

      Выполняется при показе блока, когда модификатор _visibility_visible уже установлен.

      По умолчанию проигрывает fade-in анимацию. Переопределите этот метод для реализации другой анимации.

      beforeHide

      PROTECTED

      Метод предназначен для проигрывания анимации скрытия блока.

      Выполняется перед скрытием блока. В момент вызова модификатор _visibility_visible еще не снят.

      По умолчанию проигрывает fade-out анимацию. Переопределите этот метод для реализации другой анимации.

      ВАЖНО: при реализации нестандартной анимации, необходимо вызвать callback по ее окончании.

      Параметры
      • callbackfunction

        функция, которую необходимо вызвать по окончании анимации. Не принимает аргументов и не возвращает никаких значений.

      Модификатор _autosize_yes

      Попап, автоматически подстраивающий свои размеры под оставшееся свободное место на экране в заданном направлении direction ( набор направлений задается js-параметре directions). Размещение производится с учетом отступов direction:{to:...,offset:...} и размеров хвоста

      Методы

      repaint

      Перерисовывает блок.

      При вызове направление раскрытия будет выбрано заново согласно js-параметрам блока и текущему viewport.

      Eсли блок в момент вызова скрыт, он будет показан в своей предыдущей позиции.

      Возвращаемое значение
      BEM.DOM

      getPlacingSize

      Возвращает размер области, на которую необходимо разместить попаа.

      Возвращаемое значение
      Object

      _calcPlacingSize

      Расчитывает размер области, на которую необходимо разместить попап.

      Параметры
      • directionObject

        Параметры открытия блока

      Возвращаемое значение
      Object

      Модификатор _poll_size

      Попап с динамическими размерами.

      Через определенные интервалы времени автоматически подстраивает размеры попапа под размеры контента.
