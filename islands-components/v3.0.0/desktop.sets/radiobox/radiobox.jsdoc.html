Описание

Блок radiobox – «радио-группа», состоящая из набора зависимых переключателей «радио-кнопок» (элементы блока radio).

Методы

isDisabled

Шорткат для проверки модификатора _disabled_yes.

Параметры
  • radiojQuery

    кнопка, состояние которой необнодимо проверить

Возвращаемое значение
Boolean

val

Метод можно вызывать с параметром и без. Вызвав без парамерта получаем значение аттрибута value элемента radio, соответствующего активной кнопке. Вызвав с парамертом, в котором передаем значение аттрибута value произвольного элемента radio, делаем активной кнопку, соответствующую этому элементу radio.

Параметры
  • valString

    Значение аттрибута value какого-либо элемента radio данного блока.

Возвращаемое значение
String, BEM.DOM

Аттрибут value активного элемента radio, либо объект блока

getCurrent

Метод позволяет получить активный элемент radio блока.

Возвращаемое значение
jQuery

DOM elements

uncheckAll

Метод позволяет перевести все кнопки блока в ненажатое состояние.

Возвращаемое значение
BEM.DOM

Объект блока

_getRadioByElem

STATIC

Позволяет получить элемент radio (radiobox__radio) по какому-либо потомку этого элемента в DOM-дереве.
