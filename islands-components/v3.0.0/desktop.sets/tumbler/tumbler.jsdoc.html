Описание

Тумблер.

Визуально представлен переключателем c двумя состояниями "включено"/"выключено". Предназначен для выбора одного из двух возможных значений некоторого (или произвольного) параметра. Выбор осуществляется переводом переключателя тумблера в соответствующее состояние.

Произвольные параметры можно задать в BEMJSON в полях value правого и левого элементов option. По умолчанию выбран параметр левого элемента.

Переключается кликом на тумблере, перетаскиванием переключателя мышкой и, если тумблер находится в фокусе, то клавиатурными клавишами left и right.

Методы

toggle

Переключает состояния тумблера включен-выключен, либо, если у тумблера заданы значения для переключения, переключает между левым и правым значением элемента option.

Возвращаемое значение
BEM.DOM

uncheck

Выключает тумблер либо, если у тумблера заданы значения для переключения, выбирает левый элемент option.

Возвращаемое значение
BEM.DOM

check

Включает тумблер либо, если у тумблера заданы значения для переключения, выбирает правый элемент option.

Возвращаемое значение
BEM.DOM

val

Получает/устанавливает значение тумблера. При вызове без параметров, возвращает текущее значение тумблера в виде строки. Если метод вызван со строковым параметром, то тумблер переключится в состояние, соответствующее переданному значению, если оно будет существовать среди заданных значений тумблера.

Параметры
  • valString

    Устанавливаемое значение тумблера.

Возвращаемое значение
String, BEM.DOM

Значение тумблера или объект блока.

isChecked

Получает состояние тумблера.

Возвращаемое значение
Boolean

Возвращает true, если тумблер включен, и false, если выключен.

isDisabled

Получает состояние активности тумблера.

Возвращаемое значение
Boolean

Возвращает false, если тумблер активен, и true, если не активен.

События

change

Генерируется при переключении тумблера.

Параметры
  • checkedBoolean

    Состояние тумблера, включен или выключен.

  • valueString

    Значение тумблера.
