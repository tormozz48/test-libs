    enter

    PROTECTED

    Действие при выделении варианта подсказки.

    Метод вызывается, когда пользователь выделяет вариант подсказки с помощью стрелочек вверх/вниз на клавиатуре. В блоке suggest это предусмотрено.

    Возвращаемое значение
    Boolean

    true, если есть необходимость обновить значение в поле ввода саджеста.

    select

    PROTECTED

    Действие при выборе варианта подсказки.

    Метод вызывается, когда пользователь выбирает вариант подсказки с помощью нажатия Enter или клика по нему. В блоке suggest это предусмотрено.

    Параметры
    • byKeyboardBoolean

      true, если выбор подсказки сделан с помощью клавиатуры.

    Возвращаемое значение
    Boolean

    true, если есть необходимость обновить значение в поле ввода саджеста.
