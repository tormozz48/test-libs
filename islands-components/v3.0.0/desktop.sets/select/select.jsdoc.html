Описание

Блок select(«селект») – раскрывающийся список выбора. Элемент управления имеет один уровень вложенности и позволяет выбрать только один элемент из списка.

Методы

isDisabled

Возвращает true, если блок заблокирован.

Возвращаемое значение
Boolean

isOpened

Возвращает true, если в данный момент выпадающий список элементов открыт.

Возвращаемое значение
Boolean

open

Открывает выпадающий список.

Возвращаемое значение
BEM.DOM

close

Закрывает выпадающий список.

Возвращаемое значение
BEM.DOM

val

Устанавливает значение, если передан параметр val или возвращает текущее значение, если параметр не передан.

Параметры
  • valString

    устанавливаемое значение. Будет выбран элемент __option с аттрибутом value, равным этому параметру.

Возвращаемое значение
String, BEM.DOM

текущее значение или сам блок, если значение устанавливалось.

setOptions

Устанавливает новый набор опций.

Позволяет декларативно задать список новых пунктов или групп для блока. Каждый элемент описывается в виде объекта со следующими полями:

  • item - тип элемента. Принимет следующие значения:
    • option - одиночный пункт.
    • optgroup - группа опций с дочерними элементами.
  • disabled - если true, пункт (или дочерние пункты группы) будет заблокирован и не сможет быть выбран.

Для одиночных пунктов также задаются следующие параметры:

  • value - значение опции (аттрибут value DOM-элемента). -content - отображаемое значение. Может быть как строкой, так и bemjson, при условии, что соотвествующие шаблоны доступны на клиенте.

Дополнительные параметры для групп пунктов:

  • label - заголовок группы.
  • content - массив из дочерних элементов или групп.

Примеры:

Установить набор из трех опций:

block.setOptions([
    {item: 'option', value: 'moscow', content: 'Москва'},
    {item: 'option', value: 'spb', content: 'Санкт-Петербург'},
    {item: 'option', value: 'simf', content: 'Симферополь'}
]);

Заблокированный пункт:

block.setOptions([
    {item: 'option', content: 'Нельзя выбрать', disabled: true}
]);

Группы:

block.setOptions([
    {
        item: 'optgroup',
        label: 'Россия',
        content: [
            {item: 'option', value: 'moscow', content: 'Москва'},
            {item: 'option', value: 'spb', content: 'Санкт-Петербург'}
        ]
    },
    {
        item: 'optgroup',
        label: 'Украина',
        content: [
            {item: 'option', value: 'simf', content: 'Симферополь'}
        ]
    }

]);
Параметры
  • dataObject[]

    устанавливаемые опции.

Возвращаемое значение
BEM.DOM

redraw

Перерисовывает выпадающий список.

Размеры списка будут подстроены под содержимое его элементов.

Возвращаемое значение
BEM.DOM

destruct

Уничтожает блок.

Выпадающий список будет закрыт перед уничтожением.

getDefaultParams

PROTECTED

Возвращает параметры блока по умолчанию.

Возвращаемое значение
Object

События

focus

Генерируется, когда блок получает фокус.

blur

Генерируется, когда блок теряет фокус.

change

Генерируется при смене выбранного пункта.

Параметры
  • dataObject
  • data.indexNumber

    Индекс элемента, выбранного в настоящий момент.

  • data.prevnumber

    Индекс предыдущего выбранного элемента или -1, если такого элемента нет.

Модификатор _layout_content

Селект, растягивающийся на всю ширину родителя.
