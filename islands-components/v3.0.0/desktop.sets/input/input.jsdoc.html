Описание

Поле ввода.

Оборачивает нативный контрол (input или textarea) в контейнер и реализует различные дополнения к нему.

Дополнительная подсказка.

Представляет собой контейнер (элемент __ahead) содержащий невидимую (элемент __ahead-filler) и видимую (элемент __ahead-hint) части подсказки. Невидимая часть нужна для правильного позиционирования видимой части, в ней дублируется значение из поля ввода. В видимую часть записывается остаток слова для дополнения.

Сообщение под полем ввода. Сообщения используются для показа ошибок, информации и т.д.

Смена значения в нативном контроле отслеживается с помощью polling'а его значений. Для этого используется событие tick из канала sys, реализованого в блоке i-system.

Методы

isDisabled

Проверяет, находится ли блок в отключенном состоянии. Отключенное состояние характеризуется наличием модификатора _disabled_yes.

Возвращаемое значение
Boolean

val

Устанавливает/возвращает текущее значение.

Если перадан параметр val, то устанавливает значение и возвращает экземпляр блока. При этом генерирует событие change.

Если метод вызван без параметров, то возвращает текущее значение.

Возвращает/устанавливает текущее значение

Параметры
  • valString

    Устанавливаемое значение.

  • dataObject

    Дополнительные данные, которые будут переданы в обработчик события change.

Возвращаемое значение
BEM.DOM, String

name

Возвращает имя нативного контрола.

Возвращаемое значение
String

getSelectionEnd

Возвращает позицию курсора в текстовом поле.

Возвращаемое значение
Number

destruct

Уничтожает блок.

buildAhead

PROTECTED

Создает и инициализирует контейнер для дополнительной подсказки.

Возвращаемое значение
jQuery

Контейнер дополнительной подсказки.

showAhead

PROTECTED

Устанавливает текст дополнительной подсказки если переданы параметры filler и hint, или очищает его, если параметры не переданы.

Параметры
  • fillerString

    Невидимая часть дополнительной подсказки.

  • hintString

    Видимая часть дополнительной подсказки.

Возвращаемое значение
String

Полный текст дополнительной подсказки.

clearAhead

PROTECTED

Очищает текст дополнительной подсказки.

Возвращаемое значение
Boolean

false, как знак того, что полный текст дополнительной подсказки пуст.

shiftAhead

PROTECTED

Перемещает символы из невидидимой части дополнительной подсказки в видимую ч��сть, если параметр shift положителен, и наоборот, если отрицателен.

Параметры
  • shiftNumber

    Количество символов для перемещения.

Возвращаемое значение
String

Полный текст дополнительной подсказки.

clearInput

Очищает значение в поле ввода.

Генерирует событие change.

Параметры
  • dataObject

    Дополнительные данные, которые будут переданы в обработчик события change.

Возвращаемое значение
BEM.DOM

_onSampleClick

Обработчик клика на подсказку. Подсталяет значение подсказки в поле ввода, генерирует событие change.

Параметры
  • e$.Event

    Событие клика.

События

change

Генерируется при смене значения.

focus

Генерируется при получении фокуса.

blur

Генерируется при потере фокуса.

update-items

Генерируется при обновлении саджеста новыми данными.

select

Генерируется при выборе варианта из саджеста.

Параметры
  • dataObject

    Дополнительные данные события.

  • data.itemBEM.DOM

    Блок выбранного элемента саджеста.

  • data.byKeyboardBoolean

    true, если выбор подсказки произошел с помощью клавиатуры (нажата клавиша Enter).

ahead-click
