Для того, чтобы Яндекс выглядел аккуратнее и развивался в одном стиле, были разработаны декоративные (нестандартные для операционной системы) элементы. Одним из таких элементов является кнопка. Данная кнопка предназначена только для работы с интерфейсами, она не подходит для промо-задач.
