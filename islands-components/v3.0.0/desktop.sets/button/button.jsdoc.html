Описание

Кнопка

Может быть обычной кнопкой, кнопкой действия, кнопкой-ссылкой (если указать свойство url в BEMJSON), кнопкой со счетчиком, псевдо-кнопкой, содержать иконку.

Методы

destruct

Уничтожает блок. Снимает фокус с DOM-элемента control.

События

press

Генерируется, когда кнопка нажата.

release

Генерируется, когда кнопка отпущена.

focus

Генерируется, когда кнопка получает фокус.

blur

Генерируется, когда кнопка теряет фокус.
