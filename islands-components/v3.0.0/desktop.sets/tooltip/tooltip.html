Всплывающая подсказка-указатель (tooltip)

Описание

Блок toolip предназначен для создания всплывающих подсказок.

Подсказка всегда указывает на некоторый элемент страницы owner. owner задается из JavaScript.

Подсказка скрывается по клику на ней.

Варианты использования

Объявление блока на странице

BEMJSON блока:

{
    block: 'tooltip',
    mods: {
        size: 'xs',
        theme: 'success'
    },
    content: 'Содержимое подсказки'
}

Использование в JavaScript

var tooltip = this.findBlockInside('tooltip'),
    tooltipOwner = this.findBlockInside('tooltip-owner');

tooltip
    .setOwner(tooltipOwner)    // задать объект на который будет указывать подсказка
    .setMod('shown', 'yes');   // показать подсказку

Техническое описание

Блок представляет собой обвязку над блоком popup.

Элементы

Элемент tail — хвостик подсказки. Реализован в технологии css. Размер хвостика определяется в модификаторе size.

Модификаторы

Тема оформления

Модификатор theme (обязательный, темы по умолчанию нет). Темы реализованы в технологии css.

У подсказки реализованы несколько тем:

  • normal
  • error
  • success

Размер

Модификатор size. Обязательный, размера по умолчанию нет. Размеры реализованы в технологиях css и bemhtml.

У подсказки реализован следующий набор размеров:

  • xs
  • s
  • m
  • l

Размерная сетка соответствует размерам кнопки из v3.0.0 (button).

Самозакрытие

Модификатор autoclosable.

Опциональный модификатор, реализован в технологии js. Позволяет закрывать подсказку по клику во вне ее или по нажатию клавиши ESC.

Допустимое значение:

  • yes

Состояния подсказки

  • Модификатор shown.

    • Показывает / скрывает подсказку. Допустимое значение: yes.
      Обратите внимание, прежде чем подсказка может быть показана, у нее должен быть задан owner. В противном случае, подсказка кидает JavaScript-ошибку.

  • Модификатор hovered.

    • При наведении курсора подсказка меняет прозрачность. Модификатор реализован только на уровне desktop.blocks в технологии css. Подключается по умолчанию. Допустимое значение: yes.

JS-параметры

{Number} offset – определяет отступ подсказки от элемента на который она указывает. Отступ учитывает размер хвостика __tail.
