Описание

Индикатор загрузки. Вращается посредством CSS3 анимации, в старых браузерах деградирует до CSS-sprite анимации.

JS-параметры

  • paramsObject

    Параметры блока:

  • params.animationString

    Вид анимации: 'transform' для CSS3 анимации, 'sprite' для анимации через CSS-sprite.

  • params.startDelayNumber

    Задержка (ms) перед стартом анимации.

  • params.frameAngleNumber

    Угол поворота (deg) за один кадр. Только для CSS3 анимации.

  • params.frameTimeNumber

    Время отображения (ms) одного кадра анимации.

  • params.framesNumber

    Количество кадров анимации. Только для анимации через CSS-sprite.

Методы

getDefaultParams

PROTECTED

Возвращает параметры блока по умолчанию.

destruct

PROTECTED

Уничтожает блок. Останавливает вращение.
